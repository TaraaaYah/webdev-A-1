Built mobile-first

How we built it

Engineered for the 78% of users on mobile using modern responsive web techniques.

HTML

Semantic structure

Clean, accessible HTML with a proper <meta viewport> tag ensures mobile browsers render at the correct scale — never a shrunk-down desktop layout.

CSS

Media queries & flex grids

Media queries switch from a 3-column desktop grid to a single-column mobile layout at 768px. Flexbox percentages make every element scale fluidly from iPhone to iMac.

JavaScript

Interactive calculator

Live price breakdowns, scroll-triggered animations, and a mobile hamburger menu — built without any framework, from simple hide/show to real-time arithmetic.

Design

Touch-friendly UI

Book Now buttons are a minimum of 52px tall on mobile to prevent fat-finger errors. Desktop users see more at once for confident, informed booking decisions.
